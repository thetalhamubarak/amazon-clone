# Amazon Clone

A pixel-perfect, responsive frontend clone of the Amazon homepage built using semantic HTML5 and custom CSS3. This project focuses on replicating intricate structural UI elements such as custom flexbox navigation bars, a 4-quadrant product layout grid, and authentic interactive hover animations.

## 🚀 Features
* **Authentic Navigation Bar:** Replicated navigation bar layout featuring search category drop-downs, flexible input fields, local delivery location targets, and hover-border tracking.
* **Multi-Quadrant Product Grids:** Structured signature 4-item card layout blocks for category exploration panels using explicit image grid elements.
* **Responsive Architecture:** Layout configurations built utilizing Flexbox distributions and clean CSS block spacing to adapt smoothly across viewports.
* **Interactive UI Elements:** Implemented classic gradient styling on primary call-to-action buttons along with embedded hover transitions for contextual links.

## 🛠️ Tech Stack
* **HTML5:** Semantic structural layout architecture.
* **CSS3:** Custom flexbox distributions, color variables, button gradients, and hover transitions.
* **FontAwesome:** Scalable vector icons for core UI items (search, location targets, and cart models).
